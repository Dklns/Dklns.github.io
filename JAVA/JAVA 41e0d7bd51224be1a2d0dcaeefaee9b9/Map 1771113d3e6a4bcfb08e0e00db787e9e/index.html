# Map

# 遍历

以 HashMap 为例

map 就是存储 k-v 键值对，但是为方便遍历，map 将 k-v 放在 Map.Entry 中，Map.Entry 又放在一个名叫 entrySet 的 Set 中

这个 Map.Entry 其实是编译类型，运行类型实际上是 HashMap.Node，HashMap.Node 实现了 Map.Entry，但是没有提供 getKey () 和 getValue () 方法，所以使用向上转型
