# Vue

  • 组件间传递数据

    父传子：

    子组件创建 props 对象，在 props 中写出变量详情，是否必需，数据类型等

    父组件在使用子组件的时候在标签范围内直接指定 prop=”value” ；如果想传入的值是 JavaScript 代码形式，可以使用 v-bind ，即 :prop=”value”

    子传父：

    子组件发出一个事件信号，父组件将一个 method 在这个子组件上与这个事件绑定，当这个事件触发，就调用 method

  • 简写组件发出事件信号

    当一个事件仅仅只是发出信号，或者内容很简单时，可以不用在 methods 中写方法，而是直接在标签中写

  • 计算属性相比较普通属性的反应性

    当 template 中某一处依赖 isDone 时，如果 done 发生了改变， isDone 不会改变

    但计算属性会随之改变
