# 背景图片不能完整显示

一开始背景图片就是这样

加上如下属性设置

解释一下背景尺寸的关键字

  • cover  — 浏览器将使图像足够大，使它完全覆盖了盒子区，同时仍然保持其高宽比。在这种情况下，有些图像可能会跳出盒子外
  • contain  — 浏览器将使图像的大小适合盒子内。在这种情况下，如果图像的长宽比与盒子的长宽比不同，则可能在图像的任何一边或顶部和底部出现间隙。

使用 cover 这个盒子就会比较完整的只以这一张图片作为背景，虽然会有一小部分没有被包含进盒子里

使用 contain 这个盒子确实会把整张图片作为背景，但如果取消了平铺，那么盒子会出现许多空隙

背景定位就无脑居中了

# 子盒子在弹性父盒子中的居中排布

在这个例子中我将 body 也设置为 flex 盒子，contaier 盒子的宽度是 90vw

如果默认排布，右边会有 10% 的空白，而左边没有

学到的新属性 justify-content , 设置为 center 就行了
** justify-content ** 属性定义了浏览器之间，如何分配顺着弹性容器主轴 (或者网格行轴) 的元素之间及其周围的空间。
