# 什么是 SQL

SQL (英文全称：Structured Query Language) 是结构化查询语言，专门用来访问和处理数据库的编程语言。能够让
我们以编程的形式，操作数据库里面的数据。

三个关键点：
①SQL 是一门数据库编程语言
②使用 SQL 语言编写出来的代码，叫做 SQL 语句
③SQL 语言只能在关系型数据库中使用（例如 MySQL、Oracle、SQL Server)。非关系型数据库（例如 Mongodb) 不支持 SQL 语言
