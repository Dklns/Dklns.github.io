# 如何选择 web 开发模式

不谈业务场景而盲目选择使用何种开发模式都是耍流氓。

  • 比如企业级网站，主要功能是展示而没有复杂的交互，并且需要良好的 SO, 则这时我们就需要使用服务器端渲染
  • 而类似后台管理项目，交互性比较强，不需要考虑 SO, 那么就可以使用前后端分离的开发模式。

另外，具体使用何种开发模式并不是绝对的，为了同时兼顾了首页的渲染速度和前后端分离的开发效率，一些网站采用了首屏服务器端渲染 + 其他页面前后端分离的开发模式。
