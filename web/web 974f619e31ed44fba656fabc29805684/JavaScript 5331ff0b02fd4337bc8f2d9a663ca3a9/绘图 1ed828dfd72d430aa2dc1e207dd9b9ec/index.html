# 2D 画图

在 html 中先使用一个 <canvas> 意为创建一块画布，然后在 js 中创建一个指向 <canvas> 的变量

绘制路径

arc() 函数有六个参数。前两个指定圆心的位置坐标，第三个是圆的半径，第四、五个是绘制弧的起、止角度（给定 0° 和 360° 便能绘制一个完整的圆），第六个是绘制方向（ false 是顺时针 true  是逆时针）。
